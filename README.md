This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Install

    $ git clone https://github.com/ORG/PROJECT.git
    $ cd PROJECT
    $ npm install

## Start & watch

    $ npm start

## Simple build for production

    $ npm run build
